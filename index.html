Ghostty Zsh Oh My Zsh Starship DevOps

Why this stack

Most terminal setups fail: they become overloaded or never become comfortable. This project stays in the useful middle.

Ghostty

Fast, GPU-accelerated terminal emulator with native UI, shell integration, and a modern feel.

Zsh + Oh My Zsh

A powerful shell with a practical structure for aliases, plugins, completions, and maintenance.

Starship

A prompt that is fast, portable, cross-shell, and easier to reason about than heavy theme frameworks.

Built for real work

DevOps engineers, software developers, SREs, platform engineers, and cloud specialists.

What you get

A setup that stays useful every day without unnecessary complexity.

Clean visual experience

Polished look without burying you in decoration. Readable fonts, good contrast, useful prompt content.

Portable configuration

Cleaner .zshrc and prompt config that lives in Git and moves across machines easily.

Useful context in prompt

Git branch, Kubernetes context, cloud profile, runtime versions, and command duration when relevant.

Better foundation for tools

Git, Kubernetes, Terraform, cloud CLI tools, and daily development tasks all work better.
